============ Instruction ============
1. Create a database called dynamic-galleryat phpMyAdmin and import the "images.sql" file into this database.
2. Insert some images data into the "images" table and upload the respective images to the "images/" and "thumb/" directory.
3. Open the "dbConfig.php" file and modify the $dbHost, $dbUsername, $dbPassword, and $dbName variable's value with your database credentials.
4. Open the "index.php" file on the browser ==> click on the image ==> image gallery popup will appear.
